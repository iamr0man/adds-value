- [Vue 3.2](https://vuejs.org/)

- prettier 2.5

Additionally I added [vite-svg-loader](https://github.com/jpkleemans/vite-svg-loader) for easier SVG imports.

## Install

```
git clone https://github.com/richardevcom/vue3-boilerplate.git
```

then

```
cd vue3-boilerplate
npm install
```

## Run / Develop

```

npm run dev

```

## Build

```

npm run build

```

## Preview

```

npm run preview

```

## Lint

```

npm run lint
```
